# PWA Icons

This directory needs actual icon files for the Progressive Web App to work properly.

## Required Icons:
- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

## How to Create Icons:

### Option 1: Online Generators
1. Go to https://www.pwabuilder.com/imageGenerator
2. Upload your logo or create a simple icon
3. Download the generated icons
4. Replace the placeholder files in this directory

### Option 2: Quick SVG-based Approach
Create a simple SVG with your app branding and convert to PNG at required sizes.

### Option 3: Design Tool
Use Figma, Canva, or any design tool:
- Create a 512x512 canvas
- Design your icon (use the SafeConnect theme: blue #3b82f6)
- Include a simple logo or the letter "S"
- Export as PNG
- Resize to 192x192 for the smaller version

### Suggested Icon Design:
- Background: Blue gradient (#3b82f6 to #2563eb)
- Icon: White "SC" letters or a signal/connection symbol
- Keep it simple and recognizable at small sizes
